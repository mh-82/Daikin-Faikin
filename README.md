More detailed collection on how to wire Faikin modules into Daikin ductless indoor units.

Direct connection on S403 using:

Faikin - S403

GND    - GND pin 6 or 7

12v    - +14v pin 1 (BOTTOM)

48(TX) - RX pin 5

34(RX) - TX pin 4

5v     - Not Used
